This project is used as a study case 
to a System analysis class in a Software Project discipline.



This application has the objective of transcript voice to text 
using a open source API from google to capture the audio and returning
the text in real time, this process is being broadcasted through browser
to any student in the classroom with a valid login and permission to
receive the class content, the classes will stored in a database and 
organised by the data, teacher and subject.

The first priority of the program is to help disabled 
students during class, but the development of the concept
has awaken the interest of all studies to the possibility of 
consulting the class material afterwards.


  
Being developed in group under the 
mentorship of an academic master
